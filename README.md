# 🚀 WPF-MVVM-Projects

A complete collection of modern WPF tutorials and real-world desktop application projects built using C#, WPF, MVVM, CommunityToolkit.Mvvm, SQLite, SQL Server, MahApps.Metro, Entity Framework Core, and Ollama AI.

This repository is designed for beginners to advanced developers who want to master WPF with practical examples and modern UI development techniques.
